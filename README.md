### 合并骨骼关键帧：
#### 因为千面动捕上传视频有时长限制，必须将视频分成几段；
#### 视频中人物会走动，造成上一段最后一帧和下一段第一帧之间的衔接误差。
#### 本代码按照骨骼名称、时间值、骨骼的七种变换属性对下一段动作进行校正，并合并两段动作。
#### 代码运算量大，需要运行很长时间。
